# Shopify Bottleneck Analysis Retargeting Cutdowns

## Purpose

Create short vertical follow-up assets from the result video batch for people who generated a report but did not book the paid diagnostic.

## Format

- Length: 15-30 seconds.
- Orientation: vertical.
- Captions: required.
- Sound-off safe: required.
- Focus: one diagnosis line only.
- CTA: return to the report page or book the Shopify Fix-It Diagnostic, depending on audience warmth.
