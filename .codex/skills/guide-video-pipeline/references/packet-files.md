# Content Packet Files

Use these sections as the default structure for files in `content-packets/<topic-slug>/`.

## IDEA_BACKLOG.md

Keep this at `content-packets/IDEA_BACKLOG.md`, not inside an individual packet.

```markdown
# Guide Video Idea Backlog

## Backlog

### <Idea Title>

- Status: idea
- Source packet: `content-packets/<topic-slug>`
- Working angle:

```
