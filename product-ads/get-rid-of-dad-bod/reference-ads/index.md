# Reference Ad Template Library

This folder stores uploaded/reference ads one by one. Each numbered folder contains the original image, source URL where available, and teardown notes that turn the reference into a reusable Get Rid Of Dad Bod template.

Every source-inspired template should now have a forensic teardown before layout work begins:

> `forensic-teardown-template.md`

Production asset requirements for product layers, GPT Image 2.0 backgrounds, hero images, props, and icon/sticker sets are documented in:

> `asset-requirements.md`

Default production product layer:

> `../generated/assets/product-shots/whey-pack-3quarter-ai-transparent.png`

Do not use the flat source product image as a template layer.
